# GitHub Profile Finder
GitHub Profile Finder is a simple React application that allows users to search for GitHub profiles using the GitHub API. It displays the user's avatar, name, bio, repository count, followers, and a direct link to their GitHub profile. The project demonstrates React fundamentals, API fetching, component-based architecture, and responsive UI design.

## Preview
![Application Screenshot](./screenshot/app_preview.png)

## Features
- Search any GitHub username
- Display profile picture name and bio
- View public repositories count 
- Display repositories list
- Display followers
- Direct link to GitHub profile
- Responsive design

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- GitHub REST API

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/github-profile-finder.git
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
